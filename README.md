# ⛳ Digital Heroes - Golf Charity Platform (MVP)

A modern, impact-driven subscription platform for golfers. This project focuses on a premium "Fintech" aesthetic and clear charitable transparency, moving away from traditional sports clichés.

## 🚀 Live Demo
**Project URL:** [https://golf-assignment-ashy.vercel.app/](https://golf-assignment-ashy.vercel.app/)

## ✨ Key Features & Logic
- **Rolling Score System:** Automatically manages the user's latest 5 Stableford scores (Range 1-45). New entries push out the oldest record to maintain a consistent history.
- **Charity Transparency:** Real-time calculation of the 10% minimum contribution from the subscription fee (e.g., ₹100 contribution from a ₹1000 fee).
- **Modern Dashboard:** Built with a dark-themed, high-end UI using Tailwind CSS for a professional user experience.
- **Scalable Architecture:** Structured using Next.js App Router for high performance and future multi-country support.

## 🛠️ Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
- **Version Control:** Git & GitHub
